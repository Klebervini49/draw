# Draw
## Quadro de desenhos

Projeto criado com HTML5, CCS3 E JS.

- Quadro em canvas para realização dos desenhos.
- Cores pré-selecionadas e um campo input:color para seleção de cores.
- Campo para definição do tamanho da linha.
